# Application Mobile Money

Application web de portefeuille électronique permettant des opérations financières (dépôt, retrait, transfert, épargne, paiement marchand).

## Technologies

- **Backend:** Node.js + Express + MongoDB
- **Frontend:** Angular 15+
- **Authentification:** JWT

## Structure

- /backend - API REST Node.js
- /frontend - Application Angular
- /docs - Documentation

## Installation

Voir README.md dans chaque dossier (backend/frontend) pour les instructions détaillées.

## Démarrage

### Backend
```bash
cd backend
npm run dev
```

### Frontend
```bash
cd frontend
ng serve
```
